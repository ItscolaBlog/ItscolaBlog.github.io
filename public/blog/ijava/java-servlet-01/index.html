Servlet 01

这是后端开发的开始，Servlet API，为什么是 API? 因为它的实现主要在容器中，如 Tomcat Jetty 容器等等。
我们学习 Spring 全家桶，以及各种企业级框架，那么 Servlet 就是学习它们的基础。除了能对 Java Web 后端获得基本的认识之外， 还要知道，Spring MVC 等那些大型框架，也基于 Servlet。

哪一天想自己写个 web 框架，我们也可以基于 Servlet。
