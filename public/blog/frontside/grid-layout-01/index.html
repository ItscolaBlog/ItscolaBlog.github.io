一，GridLayout

GridLayout 网格布局，实现 复杂元素排列 及 响应式，何必要用 CSS 框架。

二，Grid 外部容器基本操作

1. 定义 Grid 布局 - display: grid;

在外层容器 CSS 上加上 display: grid 以开启 Grid 网格布局。
其内部的元素将变为 Grid 布局的元素。

2. 核心概念 行和列

Grid 布局会划分多条 水平线 垂直线，交叉来划分出网格，统称为 网格线。

使用 grid-template-columns 属性，可以指定按照多少列排列，并指定每列中的元素宽度所占大小。

如图，我们指定了 4 个 100px , 则意味着划分成了 4 列进行排列。每一列元素的宽度占 100px。

当然 如果指定 3 个，就意味着分为 3 列。 有几列，看有多少个值。

当我们改为 grid-template-columns: 100px 200px 100px 100px; 第二列的元素所占宽度变为 200px。

那么，我们也可以通过 grid-template-rows 属性来指定行，我们可以指定多少行，每行占多少大小。

下面我将加更多div ，去掉 height CSS 属性 (因为要用 grid-template-rows 来指定每行大小了)。
并设定为 grid-template-rows: 100px 200px 100px 100px;

结果如图 总共 4 行，第二行所占行大小为 200px。

3. 行的大小 浮动值 与 百分比，响应式

现在有一个问题，这样指定行和列，确实很方便，很好用。 但是，最大的问题是 我们需要的是响应式。
想做出一个优质的页面，响应式是必不可少的。

我们之前介绍过 如何用 bootstrap 实现响应式。而其实，通过 Grid 布局实现响应式更为简单。

我们只需要设定 grid-template-columns 每列所占大小为浮动大小就可以了。
浮动大小为 Grid 布局中的专用单位。 以 fr 结尾。

grid-template-columns: 1fr 2fr 1fr 1fr;

意味着 第一列占总体的 五分之一 第二列占五分之二 第三列 五分之一 第四列五分之一。

除此之外，我们也可以通过设定百分比，来指定大小，实现响应式。 即所占 总体的大小。


这些单位之间也可以混合使用。 grid-template-columns: 10% 2fr 20px 1fr;
都可以。

5. 设置行和列之间的间距

我们可以通过 grid-row-gap 设置每行的间距。

也可以通过 grid-column-gap 设置列的间距。

三，Grid 布局 元素对齐方式

1. 元素横向对齐方式 justify-items

我们可以决定行和列之中的元素如何对齐。它跟 flex 布局 类似 也有主轴和纵轴。

首先我们给元素设置固定大小，实现小于网格，这样才能看出效果。 可以看到元素默认在网格的左上角对齐。

使用 justify-items 定义横向的对齐方式。 分别有 start ， end ， center ， stretch。
即 向开头对齐(左) 向结尾对齐(右) 中间对齐 拉伸占满。

比如 居中对齐。

2. 元素竖向对齐方式 align-items

使用 align-items 定义横向的对齐方式。 分别有 start ， end ， center ， stretch。
即 向开头对齐(上) 向结尾对齐(下) 中间对齐 拉伸占满。

比如 竖向居中。

四，Grid 布局 网格(纵轴和竖轴) 在整个外部容器的对齐方式

和 三 使用方式差不多，差在 三 针对容器内元素的对齐，这边针对容器的对齐。

1. 容器横向对齐方式 justify-content

属性值可以是 start | end | center | stretch | space-between; 第二章节会补充更多。
即 向开头对齐(左) 向结尾对齐(右) 中间对齐 拉伸占满 内部元素之间间隔相等。

容器相对于外面容器居中对齐。

2. 容器竖向对齐方式 align-content

属性值可以是 start | end | center | stretch | space-between; 第二章节会补充更多。
即 向开头对齐(上) 向结尾对齐(下) 中间对齐 拉伸占满 内部元素之间间隔相等。

和上面类似 不多说。
